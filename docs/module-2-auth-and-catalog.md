# Module 2: Authentication and Product Browsing

## Objective
Provide secure JWT-based sign-in and a premium storefront for exploring products.

## End-user Steps
1. Register a new customer or log in with the seeded customer account.
2. Browse the homepage and product listing page.
3. Filter by category or search by keyword.
4. Open a product detail page.
5. Add an item to cart.

## Notes
- JWT tokens are stored in local storage for demo simplicity.
- Product browsing is public; cart and order flows require authentication.
- Admin account exposes category dashboard navigation in the header.
